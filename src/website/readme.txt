1. make sure Flask is installed
2. change file paths in app/views.py
3. in terminal, cd to this folder
4. type python run.py
5. open a browser and use url localhost:5000/
6. request recommendation by localhost:5000/recommendation/<userid>
   for example, localhost:5000/recommendation/76561198158086086
                localhost:5000/recommendation/76561198074188133
