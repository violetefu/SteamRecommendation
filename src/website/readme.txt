1. make sure Flask is installed
2. in terminal, cd to this folder (./website)
3. type python run.py -u <mysql_userid> -p <mysql_passwd>
4. open a browser and use url localhost:5000/
5. request recommendation by localhost:5000/recommendation/<userid>
   for example, localhost:5000/recommendation/76561198158086086
                localhost:5000/recommendation/76561198074188133
